# Slider Example

## Instalação

1. Clone este repositório:

~~~git
https://github.com/Makson19/slider-example.git
~~~

2. Acesse a pasta do projeto:

~~~
cd slider-example
~~~

3. Instale as dependências do projeto:

~~~javascript
npm install 

ou 

yarn install
~~~

4. Execute a aplicação com o seguinte comando:

~~~javascript
npm run dev
~~~
